# KALI-NETHUNTER-KeX-ROOTLESS-FULL-REBUILD-AUDIO-BROWSER-SOUND-ONE-COMMAND-START-STOP
A complete, reproducible Kali NetHunter KeX (rootless) setup for Android with working audio and browser sound. This repository provides a fully documented rebuild process using Termux, including XFCE desktop, PulseAudio TCP bridging, Chromium audio sandbox fixes, and one-command start/stop script
